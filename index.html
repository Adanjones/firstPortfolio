    A little about Me

    The HTML5 Blank WordPress theme offers annotated support throughout the code, basic & advanced functions, very minimal mark-up and extremely neat & easy to read code.

    • WordPress

      HTML5 Blank includes advanced WordPress features and follows the WordPress Coding Standards.

    • HTML5

      Built on top of HTML5 code standards, HTML5 Blank is a lean, clean starter theme for WordPress.

    • Functions

      Pre-loaded with time saving features in functions.php, HTML5 Blank lets you ship your themes quicker.
